# Sample data

Canonical shared fixture location for repository-wide test assets.

## Conventions

- Keep shared sample files in `test/sample-data`.
- Keep the folder flat unless there is a separately agreed repository-wide change.
- Use `SampleDataFileLocator` from `test/UKHO.Search.Tests.Common` when tests need to resolve these files from output/runtime locations.
